# dual-domain-recommendation-system
Dual-domain recommendation engine that provides personalized book and movie suggestions using category filtering, genre matching, and popularity-based ranking.

Book and Movie Recommendation Engine
Overview

A dual-domain recommendation engine built in Python that recommends books and movies based on user preferences.

The system processes structured datasets and generates recommendations using category matching, genre filtering, and popularity-based ranking.

This project demonstrates foundational recommendation system design principles using data processing and ranking logic.

Problem Statement

Finding relevant books and movies from large content collections can be time-consuming.

Traditional search systems require users to manually browse through thousands of entries.

This project solves that problem by building a lightweight recommendation engine that automatically filters content based on user interests.

Features

✔ Book recommendation by category
✔ Movie recommendation by genre
✔ Popularity-based movie ranking
✔ Structured dataset processing
✔ Case-insensitive matching logic
✔ Dual recommendation pipeline
✔ Fast retrieval using dataset filtering
✔ Command-line user interaction system
